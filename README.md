# Tauri + Next.js Todo App

## About

Basic todo app that i made when learning tauri and rust.

### Testing it

1. Install dependencies:

```bash
npm install
```

2. Web:

**Next.js (Browser):**
```bash
npm run dev
```

3. Desktop:

**Tauri (Desktop App):**
```bash
npm run tauri:dev
```

4. Production build:

```bash
npm run tauri:build
```

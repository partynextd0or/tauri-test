# Tauri + Next.js Todo App

## About

Basic todo app that i made when learning tauri and rust.

### Testing it

1. Install dependencies:

```bash
npm install
```

1. Web:

**Next.js (Browser):**
```bash
npm run dev
```

2. Desktop:

**Tauri (Desktop App):**
```bash
npm run tauri:dev
```

1. Production build:

```bash
npm run tauri:build
```
